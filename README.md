# The Eagle Fitness Website

A premium, high-end gym website built with React, Vite, and Tailwind CSS.

## Features
- **Modern Luxury Design**: Black & Gold theme with glassmorphism and animations.
- **Admin Panel**: Client-side gallery management (password protected).
- **Responsive**: Fully optimized for mobile and desktop.
- **GitHub Pages Ready**: automated deployment via GitHub Actions.

## Tech Stack
- React + Vite
- Tailwind CSS
- Framer Motion
- React Router DOM
- LocalStorage (for data persistence)

## Getting Started

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Run Development Server**
    ```bash
    npm run dev
    ```

3.  **Build for Production**
    ```bash
    npm run build
    ```

## Folder Structure
- `src/components`: Reusable UI components (Navbar, Footer, Button).
- `src/pages`: Page components (Home, About, Membership, etc).
- `src/context`: React Context for state management (Gallery).
- `dist`: Production build output.
